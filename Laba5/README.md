# Лаба 5 — Телеграм-бот «Курсы валют ЦБ РФ»

Телеграм-бот для получения актуальных курсов валют ЦБ РФ прямо в чате.  
Данные загружаются с официального XML-фида Центрального Банка России — тех же, что публикует **www.rbc.ru** в разделе [«Курсы ЦБ РФ»](http://stock.rbc.ru/demo/cb.0/intraday/).

Реализован с использованием библиотеки **aiogram 3** (асинхронный Telegram Bot API фреймворк для Python; в задании упоминается как *iogram*).

---

## Функциональность

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и описание бота |
| `/rates` | Инлайн-клавиатура со всеми валютами ЦБ РФ |
| `/help` | Справка по командам |

После нажатия на кнопку с кодом валюты (например **USD**, **EUR**, **CNY**) бот выводит:
- полное название валюты;
- текущий курс к рублю;
- курс за одну единицу (если номинал > 1).

## Установка и запуск

### 1. Получите токен бота

Зайдите в [@BotFather](https://t.me/BotFather) в Telegram, создайте нового бота и скопируйте токен.

### 2. Установите зависимости

```bash
cd Laba5
pip install -r requirements.txt
```

### 3. Задайте токен

```bash
# Linux / macOS
export BOT_TOKEN="ваш_токен_здесь"

# Windows (PowerShell)
$env:BOT_TOKEN = "ваш_токен_здесь"
```

Или скопируйте `.env.example` → `.env` и вставьте токен, затем загрузите его перед запуском:

```bash
cp .env.example .env
# Отредактируйте .env, затем:
set -a && source .env && set +a   # Linux / macOS
```

### 4. Запустите бота

```bash
python bot.py
```
